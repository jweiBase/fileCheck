## 磁盘可视化工具开发计划

### 技术方案
使用 **Electron + HTML/CSS/JavaScript** 开发桌面应用，因为需要：
- 扫描磁盘文件系统
- 点击方格跳转到文件位置（需要桌面能力）

### 项目结构
```
fileCheck/
├── package.json
├── main.js          # Electron主进程
├── preload.js       # 预加载脚本
├── index.html       # 主界面
├── styles.css       # 样式
├── renderer.js      # 渲染进程逻辑
└── README.md        # 使用说明
```

### 核心功能实现
1. **磁盘扫描模块**
   - 递归扫描指定目录
   - 计算文件/文件夹大小
   - 构建文件树结构

2. **矩形可视化（Treemap风格）**
   - 主矩形代表整个磁盘
   - 根据文件大小分配区域比例
   - 大文件/文件夹内部继续嵌套分配（最多3层）
   - 不同层级使用不同颜色区分

3. **交互功能**
   - 输入框选择磁盘路径
   - 点击启动扫描按钮
   - 点击方格在资源管理器中打开文件位置
   - 悬停显示文件详细信息

4. **使用说明**
   - 在界面中集成帮助按钮和说明面板

### 实现步骤
1. 初始化Electron项目
2. 创建主进程和窗口
3. 实现磁盘扫描API
4. 实现Treemap可视化算法
5. 添加交互功能
6. 编写使用说明
